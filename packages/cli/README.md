## Installation

```
npm install -g @chaosmachine/cli
```

## Usage

```
➜ chaos --help

chaos <command>

Commands:
 chaos config  Configure the Chaos Machine
 chaos daemon  Launch the Chaos Machine daemon

Options:
 --version   Show version number                                      [boolean]
 -h, --help  Show help                                                [boolean]
```
